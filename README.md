# gonetweet

Automatic tweet / X post destruction written in go

## Installation

```shell
go install github.com/junian/gonetweet@latest
```

Create `.env` file based on `.env.example`.

```shell
cp .env.example .env
```

Fill the `.env` content based on your Twitter / X API credentials.
